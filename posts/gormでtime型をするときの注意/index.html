GORMでtime型をするときの注意

GORMで構造体をマイグレーションしたときに 以下のようにしてもtime型にはならなかった、、、

type test struct {
BreakTime                  *time.Time               `gorm:"type:time;comment:休憩時間"`
}

↓ マイグレすると

create table test
(
    break_time                   timestamp with time zone,
);

となった、

どうも、「time」のあとにスペースを入れれば解決するみたい。

type test struct {
BreakTime                  *time.Time               `gorm:"type:time ;comment:休憩時間"`
}

↓ マイグレすると

create table test
(
    break_time                   time,
);

時間がかかったけどこんな事があるのだと。。。。。
