Vite + Svelte + Tailwind CSS + daisyUI + TypeScript の環境設定

tama-tan

技術

パパッとコマンドなどだけですが、こんな感じで。。。。。。

npm create vite@latest kodawari -- --template svelte-ts

cd kodawari

npm i -D tailwindcss
npm i -D daisyui

npm install

npx tailwindcss init -p

./src/app.css に追加

@tailwind base;
@tailwind components;
@tailwind utilities;

./tailwind.config.cjs を更新

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [],
  theme: {
    extend: {},
  },
  plugins: [],
}
